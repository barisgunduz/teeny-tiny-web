### ðŸ”¹ Connecting to Your Droplet (SSH)

To connect to your Droplet, open a terminal and run:

```bash
ssh root@your-droplet-ip
```

If using an SSH key, add the `-i` flag:

```bash
ssh -i /path/to/private_key root@your-droplet-ip
```

### ðŸ”¹ Updating & Securing Your Droplet

Once logged in, update your packages:

```bash
apt update && apt upgrade -y
```

For security, create a new user:

```bash
adduser myuser
usermod -aG sudo myuser
```

Disable root SSH login:

```bash
nano /etc/ssh/sshd_config
# Change 'PermitRootLogin yes' to 'PermitRootLogin no'
systemctl restart sshd
```

### ðŸ”¹ Setting Up a Firewall

Enable **UFW (Uncomplicated Firewall)**:

```bash
ufw allow OpenSSH
ufw allow 80/tcp  # Allow HTTP traffic
ufw allow 443/tcp # Allow HTTPS traffic
ufw enable
```

### ðŸ”¹ Installing & Configuring Nginx

Install Nginx:

```bash
apt install nginx -y
systemctl start nginx
systemctl enable nginx
```

Check if Nginx is running:

```bash
systemctl status nginx
```

### ðŸ”¹ Installing Let's Encrypt SSL (HTTPS)

Install Certbot:

```bash
apt install certbot python3-certbot-nginx -y
```

Run the SSL installation:

```bash
certbot --nginx -d yourdomain.com -d www.yourdomain.com
```

Set up auto-renewal:

```bash
certbot renew --dry-run
```

### ðŸ”¹ Deploying a Node.js Application

Install Node.js:

```bash
apt install nodejs npm -y
```

Create a sample **Node.js server**:

```bash
mkdir myapp && cd myapp
nano server.js
```

ðŸ“Œ Add the following code:

```javascript

const server = http.createServer((req, res) => {
  res.writeHead(200, { "Content-Type": "text/plain" });
  res.end("Hello from DigitalOcean Droplet!");
});

server.listen(3000, () => console.log("Server running on port 3000"));
```

Run the server:

```bash
node server.js
```
