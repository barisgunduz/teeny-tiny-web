### ğŸ”¹ Pages & Routing

Next.js provides **two routing approaches**:

1ï¸âƒ£ **App Router (`app/` directory - Recommended for Next.js 13+)**  
2ï¸âƒ£ **Pages Router (`pages/` directory - Used in older versions)**  

#### ğŸ“Œ App Router (Recommended for Next.js 13+)
- Uses **Server Components by default**.
- Supports **layouts, nested routes, and async components**.

ğŸ“Œ Example: `src/app/page.tsx`
```tsx

  return <h1>Welcome to Next.js App Router!</h1>;
}
```

ğŸ“Œ Example: Dynamic Route `src/app/blog/[slug]/page.tsx`
```tsx

  return <h1>Blog Post: {params.slug}</h1>;
}
```

#### ğŸ“Œ Pages Router (`pages/` Directory - Traditional Approach)
ğŸ“Œ Example: `pages/index.tsx`
```tsx

  return <h1>Welcome to Next.js Pages Router!</h1>;
}
```

ğŸ“Œ Example: Dynamic Route `pages/blog/[slug].tsx`

### ğŸ”¹ Data Fetching in Next.js

Next.js supports different data-fetching strategies:

1ï¸âƒ£ **Server-Side Rendering (SSR)**  
```tsx
export async function getServerSideProps() {
  const res = await fetch("https://jsonplaceholder.typicode.com/posts");
  const data = await res.json();
  return { props: { data } };
}

export default function Page({ data }) {
  return <ul>{data.map((post) => <li key={post.id}>{post.title}</li>)}</ul>;
}
```

2ï¸âƒ£ **Static Site Generation (SSG)**  
```tsx
export async function getStaticProps() {
  const res = await fetch("https://jsonplaceholder.typicode.com/posts");
  const data = await res.json();
  return { props: { data } };
}
```

3ï¸âƒ£ **Incremental Static Regeneration (ISR)**
```tsx
export async function getStaticProps() {
  const res = await fetch("https://jsonplaceholder.typicode.com/posts");
  const data = await res.json();
  return { props: { data }, revalidate: 10 }; // Regenerate page every 10 seconds
}
```

4ï¸âƒ£ **Client-Side Fetching (useEffect + fetch)**
```tsx
import { useState, useEffect } from "react";

export default function Page() {
  const [data, setData] = useState([]);

  useEffect(() => {
    fetch("https://jsonplaceholder.typicode.com/posts")
      .then((res) => res.json())
      .then((data) => setData(data));
  }, []);

  return <ul>{data.map((post) => <li key={post.id}>{post.title}</li>)}</ul>;
}
```
