## ğŸ”¹ 3. Writing a Jenkinsfile (Declarative Pipeline)

ğŸ“Œ **`Jenkinsfile` (CI/CD Pipeline)**  
```groovy
pipeline {
    agent any

    stages {
        stage('Clone Repository') {
            steps {
                git 'https://github.com/your-repo/sample-app.git'
            }
        }

        stage('Install Dependencies') {
            steps {
                sh 'npm install'
            }
        }

        stage('Run Tests') {
            steps {
                sh 'npm test'
            }
        }

        stage('Build Docker Image') {
            steps {
                sh 'docker build -t my-app:latest .'
            }
        }

        stage('Deploy') {
            steps {
                sh 'docker run -d -p 3000:3000 my-app:latest'
            }
        }
    }
}
```

âœ… **This pipeline:**  
1. Clones the repo.  
2. Installs dependencies.  
3. Runs tests.  
4. Builds a Docker image.  
5. Deploys the app.
