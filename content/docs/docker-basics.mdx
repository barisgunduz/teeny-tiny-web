### ğŸ”¹ Creating a Docker Container for a Node.js App

Create a new directory:

```bash
mkdir myapp && cd myapp
```

Create a **Dockerfile**:

```dockerfile
# Use Node.js as the base image
FROM node:18

# Set the working directory
WORKDIR /app

# Copy package.json and install dependencies
COPY package.json .
RUN npm install

# Copy the entire project
COPY . .

# Expose the port
EXPOSE 3000

# Start the app
CMD ["node", "server.js"]
```

Build the Docker image:

```bash
docker build -t my-node-app .
```

Run the container:

```bash
docker run -d -p 3000:3000 my-node-app
```

### ğŸ”¹ Docker Compose

Docker Compose is used to run multi-container applications.

Create a **docker-compose.yml** file:

```yaml
version: "3"
services:
  web:
    image: nginx
    ports:
      - "80:80"
  api:
    build: .
    ports:
      - "3000:3000"
    depends_on:
      - db
  db:
    image: postgres
    environment:
      POSTGRES_USER: user
      POSTGRES_PASSWORD: password
```

Run with:

```bash
docker-compose up -d
```

### ğŸ”¹ Managing Containers & Images

List all containers:

```bash
docker ps -a
```

Stop all running containers:

```bash
docker stop $(docker ps -q)
```

Remove all containers:

```bash
docker rm $(docker ps -aq)
```

Remove all images:

```bash
docker rmi $(docker images -q)
```
